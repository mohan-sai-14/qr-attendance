Save the TU Robotics Club Logo

To use the TU Robotics Club logo in the login page, please follow these steps:

  1. Right-click on the robot logo image from the design assets
  2. Select "Save Image As"
  3. Navigate to this folder: app/client/public/
  4. Save the file with the name: robotics-logo.png
  5. Refresh the login page to see the updated logo

Note: The image should be the circular TU Robotics Club logo with the blue robot design as shown in the second image.
